Lambda on OpenStack or Function as a Service

Worked on a project to develop a serverless, event-driven, auto-scalable compute platform to execute the code on an event without getting into resource provisioning and management on the cloud. All that a user need to do is to upload a code and map it to an event that will trigger the execution. Resources are fetched or provisioned and the code is executed when an event for that code is received/occurs. It scales with every new request that comes in for a paricular function.
